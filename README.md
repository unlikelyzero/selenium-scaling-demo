# How to Run the tests

Run 1 test locally

```
npm run local-single
```

Run 10 tests locally, serially

```
npm run local10-serial
```

Run 10 tests locally in parallel

```
npm run local10
```

Run 10 tests remotely

```
npm run remote10
```

Run 100 tests remotely

```
npm run remote100
```
